# NER Datasets (Indonesian)

This directory contains Indonesian Named Entity Recognition (NER) datasets collected and organized for research and experimentation purposes.

---

## 1. NERGRIT Corpus

### Source
The dataset was obtained from the following public repository:

- Original data source:  
  https://github.com/cahya-wirawan/indonesian-language-models  
- Direct archive used:  
  `nergrit-corpus_20190726_corrected.tgz`

Original project homepage:
- https://github.com/grit-id/nergrit-corpus

### Named Entity Labels
The NERGRIT corpus contains the following named entity types:

- **CRD** — Cardinal  
- **DAT** — Date  
- **EVT** — Event  
- **FAC** — Facility  
- **GPE** — Geopolitical Entity  
- **LAW** — Law Entity (e.g., Undang-Undang)  
- **LOC** — Location  
- **MON** — Money  
- **NOR** — Political Organization  
- **ORD** — Ordinal  
- **ORG** — Organization  
- **PER** — Person  
- **PRC** — Percent  
- **PRD** — Product  
- **QTY** — Quantity  
- **REG** — Religion  
- **TIM** — Time  
- **WOA** — Work of Art  
- **LAN** — Language
